### 1. Project Overview

**AAKASH OS** is a fully browser-based Cloud Operating System that delivers a complete digital workspace inside a web browser. It provides an OS-like experience (desktop, windows, taskbar, dock, multitasking) combined with productivity tools, cloud storage, and online development environments.

The core idea is to eliminate the friction of traditional lab setups (e.g., repeated Gmail logins for Google Drive/Docs). Users log in once, work on files and projects that auto-save in the cloud, and the session automatically logs out when the tab is closed — ideal for shared computers, college labs, and temporary sessions.

**Vision:**  
A portable internet computer that feels like ChromeOS + Windows + Google Workspace, accessible from any device with a modern browser.

### 2. Core Objectives

- Provide a native OS-like user experience in the browser.
- Enable seamless productivity and development work.
- Keep initial costs near zero using generous free tiers.
- Ensure strong security and automatic session cleanup.
- Target students, developers, and users on shared/low-end machines.

**AAKASH OS — Supabase Database Schema**  
**Finalized Version 1.0**

### 1. Core Philosophy
- Use **Supabase Auth** (`auth.users`) as the source of truth for users.
- All tables in `public` schema.
- Heavy use of **Row Level Security (RLS)** — every table will enforce `auth.uid() = user_id`.
- Soft deletes where appropriate (`deleted_at`).
- Timestamps (`created_at`, `updated_at`) on all tables.
- Indexes on frequently queried columns.

---

### 2. Complete Schema (SQL)

```sql
-- Enable necessary extensions
CREATE EXTENSION IF NOT EXISTS "uuid-ossp";

-- ========================================
-- 1. PROFILES (extends auth.users)
-- ========================================
CREATE TABLE public.profiles (
    id UUID PRIMARY KEY REFERENCES auth.users(id) ON DELETE CASCADE,
    email TEXT UNIQUE NOT NULL,
    full_name TEXT,
    avatar_url TEXT,
    role TEXT DEFAULT 'user' CHECK (role IN ('user', 'student', 'admin')),
    
    storage_used BIGINT DEFAULT 0,           -- in bytes
    storage_limit BIGINT DEFAULT 104857600,  -- 100 MB default (104857600 bytes)
    
    plan_type TEXT DEFAULT 'free' CHECK (plan_type IN ('free', 'student', 'premium')),
    
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- ========================================
-- 2. FILES (Unified File System)
-- ========================================
CREATE TABLE public.files (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id UUID NOT NULL REFERENCES public.profiles(id) ON DELETE CASCADE,
    
    name TEXT NOT NULL,
    file_type TEXT NOT NULL,                    -- 'document', 'sheet', 'slide', 'note', 'image', 'pdf', 'code', etc.
    mime_type TEXT,
    
    cloudinary_public_id TEXT,                  -- Cloudinary reference
    cloudinary_url TEXT,
    file_size BIGINT NOT NULL,                  -- in bytes
    
    parent_id UUID REFERENCES public.files(id) ON DELETE CASCADE,  -- For folder structure
    is_folder BOOLEAN DEFAULT FALSE,
    
    path TEXT,                                  -- Virtual path (e.g., /Documents/Project1)
    
    metadata JSONB DEFAULT '{}',                -- Extra info (e.g., last_opened, version, etc.)
    deleted_at TIMESTAMPTZ,
    
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- ========================================
-- 3. PROJECTS (For IDEs / Online Compilers)
-- ========================================
CREATE TABLE public.projects (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id UUID NOT NULL REFERENCES public.profiles(id) ON DELETE CASCADE,
    
    name TEXT NOT NULL,
    language TEXT NOT NULL,                     -- 'python', 'c', 'cpp', 'java', 'javascript', etc.
    
    description TEXT,
    is_public BOOLEAN DEFAULT FALSE,
    
    settings JSONB DEFAULT '{}',                -- theme, font_size, etc.
    metadata JSONB DEFAULT '{}',
    
    last_run_at TIMESTAMPTZ,
    last_run_output TEXT,                       -- Optional: last execution result summary
    
    deleted_at TIMESTAMPTZ,
    
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- ========================================
-- 4. PROJECT_FILES (Files inside a project)
-- ========================================
CREATE TABLE public.project_files (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    project_id UUID NOT NULL REFERENCES public.projects(id) ON DELETE CASCADE,
    user_id UUID NOT NULL REFERENCES public.profiles(id) ON DELETE CASCADE,
    
    file_name TEXT NOT NULL,
    content TEXT,                               -- For code files (text-based)
    cloudinary_public_id TEXT,                  -- For binary assets if needed
    
    is_main BOOLEAN DEFAULT FALSE,              -- Main file to run (e.g., main.c, index.py)
    
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- ========================================
-- 5. USER_SESSIONS (Track active sessions)
-- ========================================
CREATE TABLE public.user_sessions (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id UUID NOT NULL REFERENCES public.profiles(id) ON DELETE CASCADE,
    
    session_token TEXT UNIQUE,
    device_info JSONB DEFAULT '{}',             -- browser, OS, IP hash, etc.
    started_at TIMESTAMPTZ DEFAULT NOW(),
    last_active_at TIMESTAMPTZ DEFAULT NOW(),
    
    is_active BOOLEAN DEFAULT TRUE
);

-- ========================================
-- 6. USER_PREFERENCES
-- ========================================
CREATE TABLE public.user_preferences (
    user_id UUID PRIMARY KEY REFERENCES public.profiles(id) ON DELETE CASCADE,
    
    theme TEXT DEFAULT 'dark' CHECK (theme IN ('light', 'dark', 'system')),
    wallpaper_url TEXT,
    dock_position TEXT DEFAULT 'bottom' CHECK (dock_position IN ('bottom', 'left', 'right')),
    
    desktop_layout JSONB DEFAULT '{}',          -- Position of icons on desktop
    window_preferences JSONB DEFAULT '{}',      -- Default window size, behavior
    
    notifications_enabled BOOLEAN DEFAULT TRUE,
    auto_save_enabled BOOLEAN DEFAULT TRUE,
    
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);

-- ========================================
-- 7. WORKSPACE_LAYOUTS (Optional - Save desktop state)
-- ========================================
CREATE TABLE public.workspace_layouts (
    id UUID PRIMARY KEY DEFAULT uuid_generate_v4(),
    user_id UUID NOT NULL REFERENCES public.profiles(id) ON DELETE CASCADE,
    
    name TEXT DEFAULT 'Default',
    layout_data JSONB NOT NULL,                 -- All open windows positions, sizes, z-index
    
    is_active BOOLEAN DEFAULT FALSE,
    
    created_at TIMESTAMPTZ DEFAULT NOW(),
    updated_at TIMESTAMPTZ DEFAULT NOW()
);
```

### 4. Row Level Security (RLS) Policies (Important)

```sql
-- Enable RLS on all tables
ALTER TABLE public.profiles ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.files ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.projects ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.project_files ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.user_sessions ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.user_preferences ENABLE ROW LEVEL SECURITY;
ALTER TABLE public.workspace_layouts ENABLE ROW LEVEL SECURITY;

-- Example Policies (apply similar for all tables)
CREATE POLICY "Users can only access their own data" ON public.files
    USING (auth.uid() = user_id);

CREATE POLICY "Users can only access their own projects" ON public.projects
    USING (auth.uid() = user_id);

-- Same for other tables
```

### 5. Triggers (Auto Update)

```sql
-- Auto update updated_at
CREATE OR REPLACE FUNCTION update_updated_at_column()
RETURNS TRIGGER AS $$
BEGIN
    NEW.updated_at = NOW();
    RETURN NEW;
END;
$$ language 'plpgsql';

-- Apply to all major tables
CREATE TRIGGER update_profiles_updated_at BEFORE UPDATE ON public.profiles
    FOR EACH ROW EXECUTE FUNCTION update_updated_at_column();

-- Repeat for files, projects, etc.
```
